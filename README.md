# kaprekar

This program calculates Kaprekar's constant and displays the steps in reaching it. It expects a valid 4 digit number as input.

## sample output for 1235

5321 - 1235 = 4086
8640 - 0468 = 8172
8721 - 1278 = 7443
7443 - 3447 = 3996
9963 - 3699 = 6264
6642 - 2466 = 4176
7641 - 1467 = 6174
